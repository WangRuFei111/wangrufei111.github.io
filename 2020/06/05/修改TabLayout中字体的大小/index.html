设置Tab背景:
设置Tablayout的布局模式，有两个值:
  • fixed：固定的，不能滑动，很多标签的时候会被挤压（默认是fixed）
  • scrollable：可以滑动的
设置TabLayout的布局方式，有两个值 :
  • fill：充满
  • center：居中
默认是fill，且只有当tabMode为fixed时才有效:
  1. tabMaxWidth：设置tab项最大的宽度
  2. tabContentStart：设置TabLayout开始位置的偏移量
  3. paddingStart，paddingEnd：设置整个TabLayout的内边距
  4. tabPadding，tabPaddingStart，tabPaddingEnd，tabPaddingTop，tabPaddingBottom：设置tab项的内边距
设置指示器的方位（默认指示器在下方）:
  1. top：指示器在上方
  2. bottom：指示器在下方
  3. center：指示器在中间
  4. stretch：指示器沾满整个布局
